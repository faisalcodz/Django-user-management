# Django User Management

This is a simple Django project made for learning Django fundamentals. It includes basic user authentication features.

## Features

* Signup
* Login
* Logout
* Edit Profile

## Tools Used

* Python
* Django
* HTML
* CSS
* SQLite

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Create and activate a virtual environment.
4. Install Django:

   ```bash
   pip install django
   ```
5. Run migrations:

   ```bash
   python manage.py migrate
   ```
6. Start the server:

   ```bash
   python manage.py runserver
   ```
7. Open `http://127.0.0.1:8000/` in your browser.
